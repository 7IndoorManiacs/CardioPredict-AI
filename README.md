# ❤️ CardioPredict AI

CardioPredict AI is a machine learning-powered web application that predicts the likelihood of heart disease using patient clinical data.

## Features

* Heart disease risk prediction
* Real-time probability scoring
* Color-coded risk assessment
* Flask web interface
* Bootstrap responsive UI
* Machine learning model deployment

## Dataset

UCI Heart Disease Dataset

## Machine Learning Models

* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)

## Best Model

Random Forest Classifier

ROC-AUC Score: 0.91

## Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* NumPy
* Bootstrap
* Joblib

## How To Run

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open:

http://127.0.0.1:5000
